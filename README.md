
# 📊 Sentiment Analysis Web App

A simple web application built using Flask that analyzes user input text and classifies it as Positive, Negative, or Neutral.

## 🚀 Features
- Sentiment detection
- Polarity & Subjectivity score
- Simple UI

## 🛠️ Technologies
- Python
- Flask
- TextBlob
- HTML

## 📂 Structure
sentiment-analysis-app/
│
├── app.py
├── README.md
└── templates/
    └── index.html

## ⚙️ Run
pip install  flask textblob
python -m textblob.download_corpora  
python app.py
